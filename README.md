# 📊 Data Analytics Internship - Task 2

## 📌 Project Title
Superstore Sales Dashboard - Data Visualization & Storytelling

## 📖 Objective
The objective of this project is to create meaningful and interactive visualizations that communicate business insights from sales data using Power BI.

## 🛠️ Tools Used
- Microsoft Power BI Desktop
- Superstore Sales Dataset
- GitHub

## 📂 Dataset
- Sample Superstore Dataset (CSV)

## 📈 Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Monthly Sales Trend
- Sales by Category
- Sales by Region
- Profit by Sub-Category
- Sales by Customer Segment
- Top States by Sales
- Discount vs Profit Analysis

## 📊 Business Insights
- Technology is the highest-performing category in terms of sales and profit.
- The East region generates the highest revenue.
- Consumer is the largest customer segment.
- Some sub-categories generate losses despite high sales.
- Higher discounts reduce overall profitability.
